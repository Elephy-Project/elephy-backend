# Elephy Backend

### 1. Create visual environment
`python3 -m venv env`

### 2. Activate the environment
`. env/bin/activate`

### 3. Install Flask
`pip install Flask`


## Run application
get into app directory
`cd app` then
`uvicorn main:app --reload`

## UI Interface
Visit 
`http://127.0.0.1:8000/docs/` or `http://127.0.0.1:8000/redoc`
